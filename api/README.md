# Projeto OnFly2

## Como rodar o projeto

### Opção 1: Setup automático
Para rodar o projeto execute > powershell -ExecutionPolicy Bypass -File .\setup-clean.ps1

### Opção 2: Setup manual

- na pasta api:
  - cd api
  - composer install       
  - npm install
  - php artisan serve

- na pasta front
  - cd front
  - npm install
